# minItem
[[[desc minItem

]]]
[[[desc minItem zh
计算传入数组的最小值，返回最小值所在的第一个数组元素，支持通过可选参数`getter`提取数值（或直接使用数组元素的数值）。

`getter`是[getByPath](../object/getByPath)函数的字段路径，或者回调函数，用于提取数值。
]]]
