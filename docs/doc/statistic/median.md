# median
[[[desc median

[[[desc median zh
计算传入数组的中位数，支持通过可选参数`getter`提取数值（或直接使用数组元素的数值）。

`getter`是[getByPath](../object/getByPath)的函数字段路径，或者回调函数，用于提取数值。
]]]
