# forEachFields
[[[desc forEachFields
]]]
[[[desc forEachFields zh
输入一个对象`obj`和迭代器`iterator`，遍历对象的每个字段，对每个字段的值执行`iterator`。
]]]
[[[desc forEachFields ja
オブジェクト `obj` とイテレータ関数 `iterator` を受け取り、オブジェクトの各フィールドをイテレートし、各フィールドの値に対してイテレータ関数 `iterator` を実行する関数です。
]]]
