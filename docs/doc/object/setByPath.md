[[[desc setByPath
]]]
[[[desc setByPath zh
输入一个对象`obj`，和字段所在的路径`path`，按照路径深度遍历设置对象的值`value`。
]]]

オブジェクト `obj` とフィールドパス `path` を入力し、パスに従ってオブジェクトを深くトラバースして、値 `value` を設定します。
]]]
